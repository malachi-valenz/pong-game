# Basic Pong with pygame

This pong game uses pygame to create a 2-player local game for lots of fun!
## Installation
**Python 3.6+ Required. Support only up to Python 3.13 (*Unless pygame-ce is used)***
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pygame.

```bash
pip install pygame

#OR

pip install pygame-ce
```

## Usage

```bash
#Once Installed
python pong.py
```
## Controls
|      | Player 1 | Player 2 |
|------|----------|----------|
| Up   | W        | ↑        |
| Down | S        | ↓        |


First to 3 points wins!
